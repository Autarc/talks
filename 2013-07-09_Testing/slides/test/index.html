Methodology

Test Driven Development (TDD)


      // title
      suite('My first test', function(){

        // preparation
        setup(function(){
          foo = 'bar';
        });

        // definition
        test('Check a string', function(){
          assert.equal( foo, 'bar' );
        });

      });

TDD implies agile - design - write tests - start coding

Methodology

Behavior Driven Development (BDD)


      // title
      describe('A simple test', function(){

        // preparation
        before(function(){
          arr = new Array(5);
        });

        // definition
        it('will be the size of 5', function(){
          expect( arr ).to.have.length(5);
          arr.should.have.length(5);
        });

      });

concept/idea (collaborative | != test, but behavio) - language, user stories - behavior of an applicaiton instead just functionality
